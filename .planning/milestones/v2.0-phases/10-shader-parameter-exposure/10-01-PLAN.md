  ```wgsl
  // Fresnel edge reflection (IOR-based)

  ```

  ```wgsl

  ```
- Add `envRef` to the final glass color (after the existing `coolSpec + warmSpec + rim` additions):
  ```wgsl

  ```
- IMPORTANT: Keep the existing `coolSpec` and `warmSpec` calculations using `glass.specularIntensity`. The Fresnel/env reflection is an ADDITIVE layer, not a replacement. This gives users independent control of directional specular (via `specular` prop) vs. edge-based Fresnel (via `fresnelIOR`/`fresnelExponent`/`envReflectionStrength` props).
