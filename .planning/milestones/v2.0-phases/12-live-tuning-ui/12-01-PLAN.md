  <action>
Create `demo/controls/presets.ts` containing:

1. **GlassParams interface** -- all 16 tunable fields (required, not optional). Mirror the fields from `GlassStyleProps` in `src/components/types.ts` but with all fields required (since the tuning panel always has concrete values). Import `GlassColor` from `../../src/components/types`.

2. **DEFAULTS constant** -- typed as `GlassParams`, with these values (sourced from `useGlassRegion.ts` defaults):
   - blur: 0.5, opacity: 0.05, cornerRadius: 24, refraction: 0.15
   - aberration: 3, specular: 0.2, rim: 0.15, tint: [1, 1, 1]
   - refractionMode: 'standard', morphSpeed: 8
   - contrast: 0.85, saturation: 1.4, blurRadius: 15
   - fresnelIOR: 1.5, fresnelExponent: 5.0, envReflectionStrength: 0.12
   - glareDirection: 315

3. **PRESETS** -- `Record<string, GlassParams>` with two entries:
   - `'Apple Clear Light'`: same as DEFAULTS but with tint: [0.15, 0.15, 0.2] and opacity: 0.25 (light mode defaults from useGlassRegion.ts)
   - `'Apple Clear Dark'`: same as DEFAULTS but with tint: [0.7, 0.75, 0.85] and opacity: 0.08 (dark mode defaults from useGlassRegion.ts)

4. **SECTION_KEYS** -- `Record<string, (keyof GlassParams)[]>` defining parameter grouping:
   - 'Blur & Opacity': ['blur', 'opacity', 'blurRadius']
   - 'Geometry': ['cornerRadius']
   - 'Refraction': ['refraction', 'refractionMode', 'aberration']
   - 'Lighting': ['specular', 'rim', 'fresnelIOR', 'fresnelExponent', 'envReflectionStrength', 'glareDirection']
   - 'Color Adjustment': ['tint', 'contrast', 'saturation']
   - 'Animation': ['morphSpeed']

5. **validateParams(data: unknown): GlassParams** -- merges parsed JSON with DEFAULTS. For each key in DEFAULTS: if key is 'tint', validate array of length 3 with all numbers; if key is 'refractionMode', validate 'standard' | 'prominent'; otherwise check typeof === 'number' && !isNaN. Unknown keys ignored, missing keys get defaults.

6. **exportParams(params: GlassParams): void** -- Blob + anchor download pattern:
   - `JSON.stringify(params, null, 2)` as blob with `application/json` type
   - Create anchor element, set href to `URL.createObjectURL(blob)`, download to 'glass-params.json'
   - Append to body, click, remove, revoke URL after 1s setTimeout
   - Flat JSON format per user's locked decision (no nesting, no metadata wrapper)

7. **importParams(onLoad: (params: GlassParams) => void, onError: (msg: string) => void): void** -- file picker pattern:
   - Create hidden `<input type="file" accept=".json,application/json">`
   - On change: read file with FileReader, JSON.parse, run through validateParams
   - Call onLoad with validated params on success, onError with message on failure
   - Reset `input.value = ''` after read to allow re-importing same file

Export all: GlassParams (type), DEFAULTS, PRESETS, SECTION_KEYS, validateParams, exportParams, importParams.
