  <action>
Create `pipeline/tune.ts` as the automated tuning loop entry point. This script orchestrates the full workflow: capture iOS reference, start dev server, run coordinate descent, write outputs.

**CLI argument parsing (simple, no library needed):**
- `--mode light|dark` (default: `'light'`)
- `--max-cycles N` (default: 15)
- `--threshold N` (default: 0.05)
- Parse from `process.argv` manually: scan for `--mode`, `--max-cycles`, `--threshold` flags and read the next argument as the value.

**Main function flow:**

1. **Parse CLI args** and determine mode, maxCycles, convergenceThreshold
2. **Print banner:**
   ```
   === Automated Tuning Loop ===
     Mode: {mode}
     Starting preset: {presetName}
     Max cycles: {maxCycles}
     Convergence threshold: {threshold}%
   ```
3. **Ensure output directory exists:** `pipeline/output/tuning/` (use `mkdir` with `recursive: true`). Resolve paths relative to `import.meta.dirname`.
4. **Capture iOS reference** (one-time):
   - Call `captureIOS(mode)` to get raw iOS screenshot
   - Normalize it: `normalize(iosRaw, iosNormPath)` where `iosNormPath` is `pipeline/output/tuning/ios-ref-{mode}.png`
   - Print: `  iOS reference captured and normalized`
5. **Start dev server:** `const baseUrl = await startDevServer(CONFIG.devServerPort)`
6. **Create persistent browser:** `const ctx = await createCaptureContext(mode, CONFIG.viewport)`
7. **Create scorer:** `const score = createScorer(ctx, baseUrl, iosNormPath, mode)`
8. **Get starting params:** `const startParams = { ...PRESETS[presetName] }` where `presetName = CONFIG.presets[mode]` (e.g., 'Apple Clear Light')
9. **Build tuning config:**
   ```typescript
   const config: TuningConfig = {
     params: TUNABLE_PARAMS,
     maxCycles,
     convergenceThreshold: threshold,
   };
   ```
10. **Run coordinate descent** with real-time console logging via `onIteration`:
    ```typescript
    const result = await coordinateDescent(
      startParams, config, mode, presetName, score,
      (entry) => {
        const arrow = entry.direction === 'none' ? '=' :
                      entry.direction === 'plus' ? '+' : '-';
        const delta = entry.delta < 0 ? `improved ${(-entry.delta).toFixed(3)}%` : 'no change';
        console.log(`  [${entry.cycle}/${entry.paramIndex}] ${entry.param} ${arrow} ${entry.newValue.toFixed(4)} | ${delta} | best: ${entry.bestScore.toFixed(2)}%`);
      }
    );
    ```
11. **Write outputs:**
    - `pipeline/output/tuning/best-params-{mode}.json`: `JSON.stringify(result.bestParams, null, 2)`
    - `pipeline/output/tuning/tune-log-{mode}.json`: `JSON.stringify(result, null, 2)` (full TuningResult including log array)
12. **Print summary:**
    ```
    === Tuning Complete: {mode} mode ===
      Preset: {presetName}
      Initial score: {initialScore}%
      Final score:   {finalScore}%
      Improvement:   {improvement}% ({relativeImprovement}% relative)
      Cycles:        {cycles}
      Evaluations:   {totalEvaluations}
      Elapsed:       {elapsed}
      Best params:   pipeline/output/tuning/best-params-{mode}.json
      Full log:      pipeline/output/tuning/tune-log-{mode}.json
    ```
13. **Cleanup** in a `finally` block: `closeCaptureContext(ctx)`, `stopDevServer()`
14. **Exit:** `process.exit(0)`

**Error handling:** Wrap the entire main function in try/catch. On error, print the error, run cleanup, and exit with code 1.
