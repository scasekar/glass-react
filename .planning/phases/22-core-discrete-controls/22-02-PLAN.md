CRITICAL anti-patterns from research:
- Do NOT give native <input> its own glass region (only outer GlassPanel)
- Do NOT change GlassPanel dimensions on focus (causes mask lag)
- Use CSS outline for focus ring (not shader rim alone -- WCAG SC 1.4.11)
- Set color: inherit and font: inherit on native input
