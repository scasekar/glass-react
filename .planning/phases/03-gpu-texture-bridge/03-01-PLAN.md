<objective>
Refactor BackgroundEngine from single-pass surface rendering to two-pass architecture: Pass 1 renders noise to an offscreen GPUTexture, Pass 2 blits that texture to the canvas surface.

Purpose: This is the core architectural change for Phase 3. The offscreen texture with TextureBinding usage becomes the shared texture that Phase 4 glass shaders will sample for refraction. Without this, glass components cannot access the background.

Output: Updated BackgroundEngine with offscreen texture, blit shader, blit pipeline, and two-pass render loop. Visual output unchanged from Phase 2.
</objective>

  <name>Task 1: Create blit shader and add offscreen texture + blit pipeline to BackgroundEngine</name>
  <files>engine/src/shaders/blit.wgsl.h, engine/src/background_engine.h, engine/src/background_engine.cpp</files>
  <action>
1. Create `engine/src/shaders/blit.wgsl.h` as a C++ raw string literal (same pattern as noise.wgsl.h):
   - Fullscreen triangle vertex shader (same 3-vertex pattern as noise shader)
   - UV computation with Y-flip: `uv = vec2f(position.x * 0.5 + 0.5, 1.0 - (position.y * 0.5 + 0.5))`
   - Fragment shader: `@group(0) @binding(0) var texSampler: sampler;` and `@group(0) @binding(1) var texSource: texture_2d<f32>;`
   - Fragment returns `textureSample(texSource, texSampler, uv)`
   - Store in `const char* blitShaderCode = R"(...)";`

2. Update `engine/src/background_engine.h`:
   - Rename existing pipeline members to noise-prefixed for clarity:
     - `pipeline` -> `noisePipeline`
     - `bindGroupLayout` -> `noiseBindGroupLayout`
     - `bindGroup` -> `noiseBindGroup`
     - `shaderModule` -> `noiseShaderModule`
   - Add new private members:
     - `wgpu::Texture offscreenTexture;`
     - `wgpu::TextureView offscreenTextureView;`
     - `wgpu::Sampler blitSampler;`
     - `wgpu::ShaderModule blitShaderModule;`
     - `wgpu::RenderPipeline blitPipeline;`
     - `wgpu::BindGroupLayout blitBindGroupLayout;`
     - `wgpu::BindGroup blitBindGroup;`
   - Add new private methods:
     - `void createOffscreenTexture();`
     - `void createBlitPipeline();`
     - `void createBlitBindGroup();`
   - Rename `createPipeline()` to `createNoisePipeline()` for clarity

3. Update `engine/src/background_engine.cpp`:

   a. Include `#include "shaders/blit.wgsl.h"`

   b. Rename `createPipeline()` to `createNoisePipeline()`. Inside, rename all member references to noise-prefixed names (noiseShaderModule, noiseBindGroupLayout, noisePipeline). The color target format stays as `surfaceFormat` (noise renders to offscreen texture which uses same format).

   c. Implement `createOffscreenTexture()`:
      - If `offscreenTexture` exists, call `offscreenTexture.Destroy()`
      - Create texture with: size={width, height, 1}, format=surfaceFormat, usage=RenderAttachment|TextureBinding, dimension=e2D, mipLevelCount=1, sampleCount=1
      - Create view: `offscreenTextureView = offscreenTexture.CreateView()`
      - Call `createBlitBindGroup()` at the end (bind group references the texture view)

   d. Implement `createBlitPipeline()`:
      - Create blitShaderModule from blitShaderCode using wgpu::ShaderSourceWGSL (same pattern as noise shader)
      - Create blitBindGroupLayout with 2 entries:
        - Entry 0: binding=0, visibility=Fragment, sampler.type=Filtering
        - Entry 1: binding=1, visibility=Fragment, texture.sampleType=Float, texture.viewDimension=e2D
      - Create pipeline layout from blitBindGroupLayout
      - Create blitSampler: addressModeU/V=ClampToEdge, magFilter=Linear, minFilter=Linear
      - Create blitPipeline: vertex=vs_main, fragment=fs_main, colorTarget format=surfaceFormat, topology=TriangleList, multisample count=1 mask=~0u

   e. Implement `createBlitBindGroup()`:
      - 2 entries: entry[0] binding=0 sampler=blitSampler, entry[1] binding=1 textureView=offscreenTextureView
      - Layout = blitBindGroupLayout

   f. Update `init()`:
      - Call `createNoisePipeline()` (renamed from createPipeline)
      - Call `createUniforms()`
      - Call `createBlitPipeline()`
      - Call `createOffscreenTexture()` (must be after createBlitPipeline since it calls createBlitBindGroup which needs blitBindGroupLayout)

   g. Update `render()` to two-pass architecture:
      - Keep uniform buffer write at top
      - Create single CommandEncoder
      - PASS 1 (noise -> offscreen): RenderPassColorAttachment with view=offscreenTextureView, loadOp=Clear, storeOp=Store. Set noisePipeline, noiseBindGroup, Draw(3).
      - PASS 2 (blit -> surface): Get surface texture, RenderPassColorAttachment with view=surfaceTexture.texture.CreateView(), loadOp=Clear, storeOp=Store. Set blitPipeline, blitBindGroup, Draw(3).
      - Both passes in same encoder. Single encoder.Finish() + queue.Submit().
      - Store encoder.Finish() in variable before taking address (avoid UB from Phase 2 lesson).

   h. Update `resize()`:
      - Keep surface reconfiguration
      - Add call to `createOffscreenTexture()` AFTER surface reconfiguration (recreates texture at new size + recreates blit bind group)

   i. Update `createUniforms()`: rename bindGroup/bindGroupLayout references to noiseBindGroup/noiseBindGroupLayout.
  </action>
