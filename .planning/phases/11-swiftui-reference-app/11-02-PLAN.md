<task type="auto">
  <name>Task 1: Create capture script with metadata stripping and verification</name>
  <files>
    ios-reference/capture.sh
    ios-reference/screenshots/.gitkeep
  </files>
  <action>
    Create `ios-reference/screenshots/.gitkeep` (empty file) so the output directory is tracked by git.

    Create `ios-reference/capture.sh` as an executable bash script (`chmod +x`).

    **Script interface:**
    ```bash
    ./capture.sh [variant] [mode]
    # variant: "regular" or "clear" (default: "regular")
    # mode: "light" or "dark" (default: "light")
    # Special: ./capture.sh all  -- captures all 4 permutations
    ```

    **Script structure (`set -euo pipefail`):**

    1. **Parse arguments:**
       - If `$1` is "all", loop through all 4 permutations calling self recursively
       - Otherwise, `VARIANT=${1:-regular}`, `MODE=${2:-light}`
       - Validate VARIANT is "regular" or "clear", MODE is "light" or "dark"
       - `BASENAME="${VARIANT}_${MODE}"` for file naming

    2. **Setup:**
       - `DEVICE="booted"` (assumes simulator is already running with the app)
       - `OUTPUT_DIR="$(dirname "$0")/screenshots"` (relative to script location)
       - `mkdir -p "$OUTPUT_DIR"`

    3. **Override status bar for clean captures:**
       ```bash
       xcrun simctl status_bar "$DEVICE" override \
           --time "9:41" \
           --batteryState charged \
           --batteryLevel 100 \
           --wifiBars 3 \
           --operatorName ""
       ```

    4. **Set simulator appearance:**
       ```bash
       xcrun simctl ui "$DEVICE" appearance "$MODE"
       ```
       Wait 3 seconds after appearance change for glass rendering to settle (per research: animations need time to reach steady state).

    5. **Capture 3 consecutive screenshots:**
       ```bash
       for i in 1 2 3; do
           xcrun simctl io "$DEVICE" screenshot \
               --type=png \
               --mask=ignored \
               "$OUTPUT_DIR/${BASENAME}_${i}.png"
           sleep 1  # Brief pause between captures
       done
       ```

    6. **Strip PNG metadata for clean comparison:**
       Check if `exiftool` is available. If yes, use it. If not, fall back to `sips` re-export:
       ```bash
       if command -v exiftool &>/dev/null; then
           for i in 1 2 3; do
               exiftool -all= -overwrite_original "$OUTPUT_DIR/${BASENAME}_${i}.png"
           done
       else
           echo "WARNING: exiftool not found. Install via 'brew install exiftool' for metadata stripping."
           echo "Falling back to sips re-export (may not strip all metadata)."
           for i in 1 2 3; do
               sips -s format png "$OUTPUT_DIR/${BASENAME}_${i}.png" --out "$OUTPUT_DIR/${BASENAME}_${i}.png" 2>/dev/null
           done
       fi
       ```

    7. **Verify pixel-identical reproducibility:**
       Compare all three captures byte-for-byte using `cmp -s`:
       ```bash
       if cmp -s "$OUTPUT_DIR/${BASENAME}_1.png" "$OUTPUT_DIR/${BASENAME}_2.png" && \
          cmp -s "$OUTPUT_DIR/${BASENAME}_2.png" "$OUTPUT_DIR/${BASENAME}_3.png"; then
           echo "PASS: ${BASENAME} - 3 captures are pixel-identical"
           # Keep only the first as the canonical reference
           cp "$OUTPUT_DIR/${BASENAME}_1.png" "$OUTPUT_DIR/${BASENAME}.png"
           rm -f "$OUTPUT_DIR/${BASENAME}_1.png" "$OUTPUT_DIR/${BASENAME}_2.png" "$OUTPUT_DIR/${BASENAME}_3.png"
           echo "  -> Saved: $OUTPUT_DIR/${BASENAME}.png"
       else
           echo "FAIL: ${BASENAME} - captures differ!"
           echo "  Keeping all 3 for inspection:"
           echo "    $OUTPUT_DIR/${BASENAME}_1.png"
           echo "    $OUTPUT_DIR/${BASENAME}_2.png"
           echo "    $OUTPUT_DIR/${BASENAME}_3.png"
           # Show which files differ
           cmp "$OUTPUT_DIR/${BASENAME}_1.png" "$OUTPUT_DIR/${BASENAME}_2.png" 2>&1 || true
           cmp "$OUTPUT_DIR/${BASENAME}_2.png" "$OUTPUT_DIR/${BASENAME}_3.png" 2>&1 || true
           exit 1
       fi
       ```

    8. **Cleanup status bar overrides at end:**
       ```bash
       xcrun simctl status_bar "$DEVICE" clear
       ```
       Use a `trap` to ensure cleanup runs even on error.

    **"all" mode implementation:**
    When `$1` is "all", iterate through permutations:
    ```bash
    PASS_COUNT=0
    FAIL_COUNT=0
    for variant in regular clear; do
        for mode in light dark; do
            echo "=== Capturing: ${variant} ${mode} ==="
            if "$0" "$variant" "$mode"; then
                ((PASS_COUNT++))
            else
                ((FAIL_COUNT++))
            fi
        done
    done
    echo ""
    echo "=== Results: ${PASS_COUNT} passed, ${FAIL_COUNT} failed ==="
    ```

    **Important notes:**
    - The script does NOT toggle the app's glass variant -- it captures whatever the current variant is. The user or a future automation script must set the variant via the app's in-app toggle before running capture. Document this in a comment at the top of the script.
    - The script DOES set light/dark mode via `xcrun simctl ui` because that can be controlled externally.
    - For "all" mode to work fully, the variant must be toggled in-app between regular and clear captures. Add a clear comment explaining this limitation and that Phase 14's automation will address it.
    - Add `#!/bin/bash` shebang and make executable with `chmod +x`
    - `--mask=ignored` produces a clean rectangle without the device frame/notch mask

    **IMPORTANT: The script must handle the in-app variant toggle limitation gracefully.**
    For the "all" mode, document that it captures both modes (light/dark) but the user must manually toggle the glass variant in the app between regular and clear capture sets. Print instructions to stderr when running "all" mode.
