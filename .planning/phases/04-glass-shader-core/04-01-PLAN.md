  <name>Task 1: Create glass WGSL shader</name>
  <files>engine/src/shaders/glass.wgsl.h</files>
  <action>
Create `engine/src/shaders/glass.wgsl.h` as a C++ raw string literal (same pattern as `noise.wgsl.h` and `blit.wgsl.h`). The variable name must be `glassShaderCode`.

The shader must contain:

**Vertex shader (vs_main):** Identical fullscreen triangle to blit.wgsl.h -- 3 hardcoded vertices covering clip space with UV output. Include the same Y-flip as blit: `1.0 - (clipY * 0.5 + 0.5)`.

**GlassUniforms struct (WGSL side):**
```
struct GlassUniforms {
    rect: vec4f,              // x, y, w, h in normalized [0,1] UV coords
    cornerRadius: f32,        // in pixels
    blurIntensity: f32,       // 0.0 = sharp, 1.0 = max frosted
    opacity: f32,             // 0.0 = invisible glass, 1.0 = solid tint
    refractionStrength: f32,  // UV distortion magnitude
    tint: vec3f,              // RGB tint color
    _pad: f32,
    resolution: vec2f,        // canvas pixel dimensions
    _pad2: vec2f,
};
```
Total: 64 bytes (4 x vec4f).

**Bindings:**
- @group(0) @binding(0) var texSampler: sampler;
- @group(0) @binding(1) var texBackground: texture_2d<f32>;
- @group(0) @binding(2) var<uniform> glass: GlassUniforms;

**Fragment shader (fs_main):**

1. Sample background at current UV (passthrough color for pixels outside glass).
2. Convert UV to pixel space using `glass.resolution`. Compute SDF distance from pixel to glass rect center using `sdRoundedBox` (Inigo Quilez formula: `let q = abs(p) - b + r; return min(max(q.x, q.y), 0.0) + length(max(q, vec2f(0.0))) - r;`).
3. Anti-alias the SDF mask: `let w = fwidth(dist); let mask = smoothstep(w, -w, dist);` -- 1.0 inside, 0.0 outside, smooth transition at edge.
4. Early return bgColor if mask < 0.001 (outside glass region).
5. Inside glass region: compute distorted UV using barrel distortion from glass center: `let offset = uv - glassCenter; let d = length(offset / (glass.rect.zw * 0.5)); let distortedUV = uv + offset * glass.refractionStrength * (1.0 - d);`
6. 9-tap (3x3) weighted blur sampling at distorted UV using compile-time constant loop bounds (`for (var x = -1; x <= 1; x++)` nested with `for (var y = -1; y <= 1; y++)`). Weight: `1.0 / (1.0 + f32(x*x + y*y))`. Blur radius = `glass.blurIntensity * 4.0` texels. Use `textureSample` (valid because loop bounds are compile-time constants -- uniform control flow).
7. Mix blurred color with tint: `mix(blurColor, vec4f(glass.tint, 1.0), glass.opacity)`.
8. Composite glass over background using mask: `mix(bgColor, glassColor, mask)`.

IMPORTANT: Do NOT use `textureSampleLevel` -- use `textureSample` since all sampling is in uniform control flow.
