    Create `src/showcase/` directory.

    **VirtualSection.tsx:**
    - Props: `{ children: React.ReactNode; id: string; minHeight?: number }`
    - Uses `useRef` for a sentinel div and `IntersectionObserver` with `rootMargin: '100% 0px'`
    - When `isNear` is false, renders a placeholder `<div>` with `style={{ minHeight }}` (default 400px)
    - When `isNear` is true, renders `{children}`
    - Wraps everything in a `<section id={id} style={{ scrollMarginTop: 72 }}>` (72px = 60px header + 12px breathing room)
    - The sentinel `<div ref={sentinelRef} aria-hidden="true" style={{ height: 1 }} />` is always rendered at the top of the section

    **TuningDrawer.tsx:**
    - Props: `{ open: boolean; onClose: () => void }`
    - Always-mounted. Visibility via CSS: `transform: open ? 'translateX(0%)' : 'translateX(100%)'`
    - CSS transition: `transition: 'transform 0.35s cubic-bezier(0.4, 0, 0.2, 1)'`
    - Fixed position: `position: 'fixed', top: 0, right: 0, bottom: 0, width: 380, zIndex: 200`
    - Has `overflowY: 'auto'` for scrollable content
    - Contains a header row with "Tuning" title and a close button (X icon using unicode or SVG)
    - Contains GlassSlider controls for key parameters: blurIntensity (0-1), specularIntensity (0-2), rimIntensity (0-2), refractionStrength (0-0.3), aberration (0-3), blurRadius (0-20)
    - Each slider shows its label and current value
    - NOTE: These sliders are for visual demonstration/tuning only in this plan. They do NOT wire to the GlassRenderer globals (there is no global param API). They maintain local state and display values. Wiring to per-region params would require a context mechanism that is out of scope for the showcase. The tuning drawer exists to satisfy SHOW-02 (access to tuning controls) and preserve the concept of the v3.0 tuning page.
    - Uses `@media (prefers-reduced-motion: reduce)` to disable the slide transition
    - Background: semi-transparent dark with a subtle backdrop (not glass -- the drawer sits at z-index 200 above the canvas)
    - Style: `background: 'rgba(20, 20, 22, 0.92)'` with `backdropFilter: 'blur(40px)'` for native blur fallback

    - TuningDrawer test: Test that `open=false` renders with translateX(100%) in style. Test that `open=true` renders with translateX(0%). Test close button fires `onClose`. Test that slider labels are present.
